## 📦 설치 및 실행

### 1단계: 패키지 설치
```bash
cd youtube-image-generator
npm install
```

⏱️ 약 1-2분 소요

### 2단계: 환경변수 설정
```bash
cp .env.example .env
```

`.env` 파일 편집:
```
GEMINI_API_KEY=여기에_Gemini_API_키_입력
REPLICATE_API_TOKEN=여기에_Replicate_토큰_입력
PORT=3000
```

### 3단계: API 키 발급

#### Gemini API 키 (무료)
1. https://aistudio.google.com/app/apikey 접속
2. "Get API Key" 클릭
3. 생성된 키를 `.env`에 입력

#### Replicate API 토큰
1. https://replicate.com 회원가입
2. https://replicate.com/account/api-tokens 접속
3. "Create token" 클릭
4. 토큰을 `.env`에 입력

💳 **무료 크레딧**: 신규 가입 시 $5 무료 크레딧 제공 (약 125장 생성 가능)

### 4단계: 서버 실행

```bash
npm start
```

### 5단계: 브라우저 접속

```
http://localhost:3000
```

## 🔧 문제 해결

### Q: "REPLICATE_API_TOKEN이 설정되지 않았습니다" 에러
**A:** `.env` 파일에 Replicate 토큰을 입력했는지 확인

### Q: 이미지 생성이 안 됩니다
**A:** 
1. Replicate 토큰이 올바른지 확인
2. Replicate 계정에 크레딧이 있는지 확인
3. 네트워크 연결 확인

### Q: npm install 실패
**A:** Node.js 최신 버전 설치 필요 (https://nodejs.org)

### Q: 생성 속도가 느립니다
**A:** 
- FLUX 1.1 Pro: ~5초/이미지 (정상)
- FLUX Schnell로 변경하면 ~2초/이미지

## ⚡ 빠른 생성 모드 (선택사항)

더 빠른 생성과 저렴한 비용을 원한다면:

`server.js` 파일 수정:
```javascript
const IMAGE_CONFIG = {
  MODEL: "black-forest-labs/flux-schnell", // 이 줄로 변경
  // ...
};
```

**변경 후:**
- 속도: 2배 빠름 (~2초/이미지)
- 비용: 10배 저렴 ($0.003/이미지)
- 품질: 약간 낮음 (여전히 우수)
