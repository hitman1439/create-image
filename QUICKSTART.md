# 🚀 빠른 시작 가이드

## 📦 설치 및 실행 (3분 완성!)

### 1단계: 프로젝트 다운로드
이 폴더를 원하는 위치에 저장하세요.

### 2단계: 의존성 설치
터미널(또는 명령 프롬프트)을 열고 프로젝트 폴더로 이동:

```bash
cd youtube-image-generator
npm install
```

⏱️ 약 1-2분 소요

### 3단계: API 키 설정

1. **Gemini API 키 발급** (무료)
   - https://aistudio.google.com/app/apikey 접속
   - "Get API Key" 버튼 클릭
   - API 키 복사

2. **Replicate API 토큰 발급** (이미지 생성용)
   - https://replicate.com 회원가입
   - https://replicate.com/account/api-tokens 접속
   - "Create token" 클릭
   - 토큰 복사

3. **환경변수 파일 생성**
   ```bash
   # .env.example을 복사하여 .env 파일 생성
   cp .env.example .env
   ```

4. **API 키 입력**
   `.env` 파일을 텍스트 에디터로 열고:
   ```
   GEMINI_API_KEY=여기에_Gemini_API_키_붙여넣기
   REPLICATE_API_TOKEN=여기에_Replicate_토큰_붙여넣기
   PORT=3000
   ```

### 4단계: 서버 실행

```bash
npm start
```

### 5단계: 브라우저에서 열기

```
http://localhost:3000
```

## 📞 문제 해결

### Q: npm install이 안 돼요
A: Node.js 최신 버전 설치 필요 (https://nodejs.org)

### Q: API 키 오류가 나요
A: .env 파일의 API 키 확인 및 따옴표 제거

### Q: 포트 3000이 이미 사용중
A: .env에서 PORT=4000으로 변경
