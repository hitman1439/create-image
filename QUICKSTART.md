# 🚀 빠른 설정 가이드 (3분 완성!)

## ✅ v2.0 주요 변경사항

### 1. **병렬 비동기 처리 적용**
- 10개 이미지를 동시에 생성
- 생성 시간 약 4배 향상 (150초 → 40초)

### 2. **실시간 이미지 표출**
- 생성되는 즉시 각 장면 카드에 표시
- 기다림 없는 즉각적인 피드백

### 3. **개별 다운로드 기능**
- 각 이미지마다 다운로드 버튼
- ZIP 압축 외에도 선택적 다운로드 가능

## 📦 설치 및 실행

### 1단계: 패키지 설치
```bash
cd youtube-image-generator
npm install
```

### 2단계: 환경변수 설정
```bash
cp .env.example .env
```

`.env` 파일 편집:
```
GEMINI_API_KEY=여기에_API_키_입력
PORT=3000
```

### 3단계: Gemini API 키 발급
1. https://aistudio.google.com/app/apikey 접속
2. "Get API Key" 클릭
3. 생성된 키를 `.env`에 입력

### 4단계: 서버 실행
```bash
npm start
```

### 5단계: 브라우저 접속
```
http://localhost:3000
```

## 🎯 사용 흐름

1. **대본 입력** → 텍스트 영역에 대본 입력
2. **분석** → 10개 장면으로 자동 분할
3. **이미지 생성** → 병렬 방식으로 동시 생성
4. **실시간 확인** → 생성 즉시 각 장면에 이미지 표시
5. **개별 다운로드** → 원하는 이미지만 선택 다운로드
6. **전체 다운로드** → ZIP으로 모든 이미지 다운로드

## 🔧 문제 해결

### Q: "GEMINI_API_KEY가 설정되지 않았습니다" 에러
A: `.env` 파일에 API 키를 입력했는지 확인

### Q: 이미지 생성이 안 됩니다
A: 
1. API 키가 올바른지 확인
2. Google AI Studio에서 Gemini API 활성화 확인
3. 네트워크 연결 확인

### Q: 병렬 처리 중 일부만 성공합니다
A: 정상입니다. 성공한 이미지만 다운로드하고, 실패한 장면은 재시도할 수 있습니다.
